# 开发工具
开发工具为eclipse
开发语言为Java
借助了eclipse里的Windows builder插件
# 运行环境
windows
# 功能描述
输入年份，计算机字长，程序员每天工作量，程序员每月工资，可以计算出计算机的存储容量，存储器价格以及计算机装满整个存储器成本（程序员每个月按照工作30天进行计算）
