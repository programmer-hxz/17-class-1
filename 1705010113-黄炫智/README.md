功能描述
----------------------------
运行后可输入四个数，分别是年份、月薪、机器字长、一天开发指令数。
通过输入的四个数使用公式计算出题目所需求的：
	①存储容量需求M = 4080*e^0.28(年份-1960)
	②存储器价格P = 0.003*字长*0.72^(年份-1974)*M
	③需要工作量N = M/(指令数*20)
	④指令成本S = N*月薪
ps：默认每月二十个工作日
